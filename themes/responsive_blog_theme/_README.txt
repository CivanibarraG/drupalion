
-- Responsive blog theme --

1. Go to Appearance > Settings.
	Standard layout: 
		Set the width of each sidebar: Unit: px 370 370
		Set the page width: Unit: % 100
		Do not set a max width
		Standard Screen Media Queries: only screen and (min-width:1170px)

	Tablet Layout: 
	- Landscape tablet:
		Set the width of each sidebar: Unit: % 30 30
		Set the page width: Unit: % 100
		Tablet Landscape Media Queries: only screen and (min-width:724px) and (max-width:1169px)
	- Portrait tablet:
		Set the page width: Unit: % 100
		Tablet Portrait Media Queries: only screen and (min-width:481px) and (max-width:723px)

	Smartphone Layout: 
	- Landscape smartphone:
		Smartphone Landscape Media Queries: only screen and (min-width:321px) and (max-width:480px)
	- Portrait smartphone:
		Smartphone Portrait Media Queries: only screen and (max-width:320px)

Any problems please post an issue in the Responsive blog theme issue queue on Drupal.org:
https://www.drupal.org/project/issues/responsive_blog_theme
